◆ c_os_task_schedule()

void c_os_task_schedule ( void )

Schedules the next task to run in the operating system.

This function is responsible for selecting and switching to the next ready task in the cooperative operating system kernel. It manages task states and ensures that the highest-priority ready task is executed.

Note
This function should be called periodically or when a task yields control to the scheduler.
