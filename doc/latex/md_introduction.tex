Cordelia Real-\/\+Time Operating System (CRTOS) is a lightweight, educational RTOS implementation for ARM Cortex-\/\+M3/\+M4 microcontrollers.\hypertarget{md_introduction_autotoc_md0}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{md_introduction_autotoc_md0}
CRTOS provides essential real-\/time operating system functionality with a educational value. It implements preemptive multitasking using round-\/robin scheduling and hardware-\/specific context switching for ARM Cortex-\/\+M4 processors.
